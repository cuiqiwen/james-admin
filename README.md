### 业务返回码

业务的返回代码统一使用 项目定义的错误码。

- A00000：正确返回
- E00001: 通用错误
- E10001: X-Token Header 不存在
- E10002: X-Token 信息已过期
- E10003: 用户信息不存在
- E10004: 无权限操作
